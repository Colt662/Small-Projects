# Small Projects / School Assignments 
Various basic projects of mine in different languages done for Navarro College

C projects are the oldest

PHP and most of the Java projects are the most recent

## How to Run
### C#
1. Install a C#  IDE. The IDE I use is [JetBrain Rider](https://www.jetbrains.com/rider/).
2. Download the .cs, .csproj, and .sln files for the project, and place them together in a folder.
3. Use your IDE (I have only verified Rider) and open the folder.
4. Run the project through your IDE. Any extra steps should be taken care of by your IDE.

### C
1. Install a C IDE. The IDE U use is [Dev C++](https://sourceforge.net/projects/orwelldevcpp/).
2. Download the .c files for the project, and place them together in a folder.
3. Use your IDE (I have only verified Dev C++) and open the main file.
4. Run the project through your IDE.

### HTML/CSS/JavaScript
1. Download the ZIP file for the whole repository and un-zip it to a folder.
2. Navigate to the HTML project you want to run and double click the index.html file. Select your preferred browser if you are asked how you want to open the file.
3. Navigate to the different site pages in your browser.

### Java
1. Install a Java  IDE. The IDE I use is [JetBrain IntelliJ](https://www.jetbrains.com/idea/).
2. Download the .java files for the project, and place them together in a folder.
3. Use your IDE (I have only verified IntelliJ) and open the folder.
4. Open the main file and run it through your IDE.

### PHP
1. Install [Xampp](https://www.apachefriends.org/download.html).
2. Download the ZIP file for the whole repository and un-zip it to a folder.
3. Drag the folder for the PHP project into C:\xampp\htdocs.
4. Run Xampp and start an Apache server.
5. Type into your browser "http://localhost/" followed by the name of the project folder eg. http://localhost/File Input-Output
